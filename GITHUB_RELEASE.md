# Codex 额度小窗 v1.0.0-10s

这个版本继续更新旧版 `Codex额度小窗`，重点是降低置顶检测频率。

## 更新内容

- 置顶检测从 2 秒调整为 10 秒。
- 保留任务栏最左侧贴靠能力。
- 保留窗口位置、宽度、高度编辑。
- 保留单行额度展示。
- 保留托盘菜单、点击穿透和一直置顶。

## 下载

请下载：

```text
Codex额度小窗-1.0.0-portable-10s.exe
```

## 使用

双击 exe 运行。右键托盘图标可以刷新额度、吸附到任务栏左侧、编辑位置、调整设置或退出。

## 注意

当前构建未做代码签名，Windows SmartScreen 可能会提示未知发布者，这是未签名 exe 的常见现象。
